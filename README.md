# Antithesis Aether MK1



## Description

The Antithesis Aether is a parallel development of the original. It heavily uses 3D printed parts, drastically reducing cost and complexity. It currently has two major options. An all plastic version in which everything, gantry included is plastic. The main drawback with the plastic gantry version is a reduction in rigidity. This will limit acceleration due to excessive deflection. The second option has an aluminum gantry plate, bed bracket, and uses an extrusion on the X axis. It has the same rigidity on the XY as the MK3.1 and still benefits from a reduction in cost and complexity. It also demonstrates identical print quality to the MK3.1.



Currently the Aether is still in development and it hasn't been tested to the extent of the MK3.1. As such you may encounter issues and oversights. Everything including the CAD and BOM is subject to change daily or weekly. There probably won't be major changes to the BOM at this point but expect minor changes. The github will be updated with more documentation in the future but will be fairly sparse for now.

## Printed Parts

I would recommend printing the parts out of ABS or ASA with the following settings: 3 walls, 40% Gyroid infill. The only exception is the plastic gantry plate and bed bracket. Those should be printed with as high an infill percentage as possible. I find 75% works pretty well. Ignore this if using the metal gantry plate and bed bracket. You may also want to use mouse ears on some parts if warping is an issue. At least one person has printed theirs out of PLA and has not encountered issues yet.
